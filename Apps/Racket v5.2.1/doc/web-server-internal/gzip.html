5.8 GZip

 (require web-server/private/gzip)
The Web Server provides a thin wrapper around file/gzip and file/gunzip.

(gzip/bytes ib) → bytes?
  ib : bytes?
GZips ib and returns the result.

(gunzip/bytes ib) → bytes?
  ib : bytes?
GUnzips ib and returns the result.
